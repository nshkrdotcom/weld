# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-26

### Added
- Initial release.
- Manifest-driven projection loading for multi-Mix monorepos.
- Source assembly into generated standalone Hex package directories.
- Strict audit mode for app-identity-sensitive bundle checks.
- `mix weld.audit`, `mix weld.build`, and `mix weld.verify` task entrypoints.
